# 起立促しアプリ 実施計画書

## 目標
設定した時間（初期設定1時間）ごとに起立を促す、シングルファイルのHTMLアプリケーションを作成します。このアプリは、GitHub Pagesで簡単に公開できるように設計され、美しくプレミアムなデザインを備えています。言語はすべて日本語です。

## ユーザー確認事項
- 特になし。

## 変更内容

### ルートディレクトリ
#### [NEW] [standup-reminder/index.html](file:///c:/git/CLI_test/standup-reminder/index.html)
- **構造**: セマンティックHTML5。
- **スタイル**: 内部 `<style>` ブロック。
    - テーマ: モダンなダーク/グラデーションテーマ。
    - コンポーネント: グラスモーフィズム（すりガラス風）のカード、円形プログレスバー（SVG）、滑らかなトランジション。
    - フォント: Google Fonts (Noto Sans JP / Outfit)。
- **ロジック**: 内部 `<script>` ブロック。
    - `StandUpTimer` クラスによるカウントダウン制御。
    - 通知APIを使用したブラウザ通知および Web Audio API によるアラート音（ループ再生）。
    - ユーザーが停止するまでアラーム鳴動を継続。
    - LocalStorage による設定（通知間隔）の保存。

## デザイン仕様
- **ビジュアル**: 深い紺色と紫のグラデーション背景。背後で動くオーブ。
- **タイポグラフィ**: 清潔感があり読みやすいサンセリフ体。
- **アニメーション**: タイマー終了時の視覚効果、プログレスリングの滑らかな更新。
- **インタラクション**: ボタンのホバーエフェクト、スタート/一時停止の明確な状態表現。

## 検証計画
### 手動検証
1. ブラウザでファイルを開く。
2. デフォルト時間が60分であることを確認。
3. タイマーを開始し、カウントダウンを確認。
4. 設定を短い時間（例: 1分）に変更し、通知が届くか確認。
5. ブラウザの通知許可要求が正しく行われるか確認。
6. 「リセット」機能の動作確認。
