# 起立促しアプリ ウォークスルー

GitHub Pagesにアップロード可能な、シングルファイルの起立促しアプリケーションです。

## 特徴
- **設定可能なタイマー**: 初期設定は60分ですが、設定画面から自由に変更可能です。
- **プレミアムなデザイン**: グラスモーフィズム（すりガラス風）を採用し、背景にはアニメーションするオーブを配置したモダンなダークテーマです。
- **多角的な通知**:
    - **視覚的通知**: カウントダウン終了時にプログレスリングが緑色に変わります。
    - **ブラウザ通知**: OS標準の通知を表示します。
    - **音声通知**: Web Audio APIを使用し、ユーザーが停止するまでビープ音を繰り返し鳴らします。
    - **即時停止**: アラーム中は「アラーム停止」ボタンが表示されます。
- **設定の永続化**: LocalStorageを使用しているため、ページをリロードしても通知間隔の設定が保持されます。
- **完全日本語化**: UI、メッセージ、タイトル、通知内容のすべてが日本語です。

## ファイルの構成
- `index.html`: アプリケーション本体（HTML/CSS/JS内蔵）
- `implementation_plan.md`: 設計・実施計画
- `README.md`: 本ドキュメント

## GitHub Pagesへのデプロイ方法
1. GitHubリポジトリに `standup-reminder` フォルダをプッシュします。
2. リポジトリの [Settings] -> [Pages] を開きます。
3. デプロイ元のブランチとディレクトリを選択します（例: `main` ブランチ、`/root`）。
4. 設定を保存し、数分待つと公開用URLが発行されます。

## 動作確認済み項目
- **タイマー**: カウントダウンが正しく行われること。
- **リセット**: タイマーが初期値に戻ること。
- **設定**: 通知間隔を変更し、保存・反映されること。
- **レスポンシブ**: スマートフォンやPCなど、画面サイズに合わせて表示が調整されること。

ブラウザで `index.html` を直接開くことで、すぐに動作を確認いただけます。
